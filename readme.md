A command line minigame learning project.

Make flavorful dinners for the dinosaur market and earn as many coins as you can!

### Playing

You must have the [Deno](https://deno.land) runtime to play.

1. Clone this repo
2. cd into the directory
3. Run:

```
deno run index.ts
```
